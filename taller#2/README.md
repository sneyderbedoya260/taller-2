## Descripcion General

Este taller cubre los fundamentos de la programacion en Python a traves de 26 ejercicios organizados en 6 secciones tematicas. Cada algoritmo fue desarrollado aplicando buenas practicas de codificacion, validacion de entradas y estructura clara.

## Como Ejecutar

1. Asegurarse de tener Python 3.8 o superior instalado.
2. Instalar dependencias:
```
pip install -r requirements.txt
```
3. Ejecutar cualquier algoritmo desde la raiz del proyecto:
```
python taller/seccion1/algoritmo_1.1.py
```
